# 🎙️ Srishti AI – Voice-Controlled Personal Assistant

Srishti AI is a Python-based voice assistant that automates desktop tasks using natural language commands. It helps you open applications, get reminders, and interact with your computer using voice.

## 🚀 Features
- 🎤 Wake word detection and continuous listening
- 🗣️ Responds to multiple commands (e.g., open apps, search online)
- 🔊 Text-to-speech interaction with system tray support
- 🧠 Built with minimal memory usage and fast response time

## 🔧 Tech Stack
- **Language**: Python
- **Libraries**: SpeechRecognition, pyttsx3, pyaudio, os, subprocess
- **Platform**: Windows (system automation)
- **Others**: Tray icon integration, background execution

## 💻 How to Run

```bash
git clone https://github.com/Whiteroot09/srishti-ai-assistant.git
cd srishti-ai-assistant
pip install -r requirements.txt
python srishti.py
```
